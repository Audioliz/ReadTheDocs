Rapports
========

Les rapports vous donnent une vue globale de vos données.

- **BI** : rapports interactifs préconfigurés avec des indicateurs clés et des KPIs
- **Lizz** : discutez avec vos données
- **Usage** : rapport d'utilisation Audioliz
- **Tableaux de bord** : tableaux de bord personnalisables qui peuvent être partagés entre les utilisateurs.
