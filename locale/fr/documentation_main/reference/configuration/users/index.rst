Utilisateurs
============

Le type de groupe d'utilisateurs définit le périmètre auquel un utilisateur peut accéder :
|52| - |$|AUDIOLIZ|$| : tous les appels|52|
 - |$|MAIN|$| : appels du client
 - |$|SUB|$| : appels du groupe d'utilisateurs

Le rôle de l'utilisateur définit ce qu'il peut faire sur ce périmètre.
